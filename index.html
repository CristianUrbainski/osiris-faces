Bem-vindo ao Osiris-Faces

Osiris-Faces é um projeto de componentes JSF com a identidade visual do Twitter Bootstrap para que seja mais bonito aos olhos do usuário, você pode fazer um fork do projeto e conhece-lo, e passar a usa-lo nos seus projetos, e está livre para criar mais componentes e ajudar outros desenvolvedores de software com suas contribuições no projeto, este projeto nasceu durante o meu projeto de TCC, quando eu quis alguns componentes visualmente mais bonitos do que aqueles que já existiam e decidi criar meus próprios componentes.

Usando o Osiris-Faces

Depois de estar com a lib do Osiris-Faces no classpath do seu projeto web, você pode começar a usa-lo, primeiro você deve adicionar o namespace do projeto a sua página xhtml, desta maneira:

xmlns:osiris="http://osiris.faces.com/ui/bootstrap"

Depois disso você já terá todos os componentes desta biblioteca disponíveis para você, para a exemplificação dos componentes disponíveis há um projeto no github disponível para que vocês novos usuários do Osiris-Faces possam ter exemplos reais em um sistema de como utilizar os componentes, o projeto está disponível em:

https://github.com/CristianUrbainski/cdi-teste/

Vocês podem fazer o fork do projeto e roda-lo na sua máquina, apenas lembre-se de usar o servidor web Glassfish para isso, lá haverá exemplos de todos os componentes, eu ainda estou desenvolvendo o projeto, se precisarem de algum exemplo que ainda não exista no projeto podem falar comigo.
